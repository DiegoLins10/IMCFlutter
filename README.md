### Resumo
Um aplicativo usando flutter que calcula o Índice de Massa Corporal (IMC) que é um parâmetro bastante utilizado para classificar o indivíduo de acordo com seu peso, altura e sexo.

### Tecnologias
- Flutter
- Vs code
- Dart
